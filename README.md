# Clase03_POOII
Repositorio de la Unidad 2 – Tema 3 de POO II, enfocado en la Introducción a ADO.NET en ASP.NET MVC. Incluye el uso de proveedores de datos, objetos como SqlConnection, Command, DataAdapter y DataReader, y la gestión de cadenas de conexión en web.config para trabajar con SQL Server.
